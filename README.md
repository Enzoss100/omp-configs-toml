# Oh-my-posh Configs

This repository stores all my favorite Oh-my-posh configurations (custom and themed from the omp repository)

## Contents
- Default config
- Personal config
- Themes

## Prerequisites
- Install oh-my-posh using curl
- Install unzip
- create a directory for your configurations under `.config/`
